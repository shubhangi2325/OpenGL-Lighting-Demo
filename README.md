# OpenGL Per-Fragment Phong Lighting on Sphere

This project demonstrates the implementation of the Phong lighting model using the OpenGL programmable pipeline. A 3D sphere is rendered with realistic lighting using GLSL shaders where lighting calculations are performed per-fragment.

The application creates a Win32 window and initializes the OpenGL rendering context without using external frameworks such as GLFW or GLUT.

## Features

• Rendering of a 3D Sphere  
• Per-fragment Phong lighting implementation  
• Vertex and Fragment shaders using GLSL  
• Ambient, Diffuse and Specular lighting components  
• Model, View and Projection transformations  
• GPU buffer management using VAO and VBO  
• Real-time lighting calculation

• Per-vertex vs per-fragment lighting
